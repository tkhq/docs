<H3Bordered text="Authorizations" />

<ParamField header="X-Auth-Proxy-Config-Id" type="string" required={true}>
  Your Auth Proxy config ID, found in **Dashboard → AUTH**. See [Auth Proxy reference](/reference/auth-proxy) for setup.
</ParamField>


<H3Bordered text="Response" />
A successful response returns the following fields:

<ResponseField name="enabledProviders" type="array" required={true}>
  List of enabled authentication providers (e.g., 'facebook', 'google', 'apple', 'email', 'sms', 'passkey', 'wallet')
  <Expandable title="enabledProviders details">
    <NestedParam parentKey="enabledProviders" childKey="item" type="string" required={true}>
item field
</NestedParam>

  </Expandable>
</ResponseField>
<ResponseField name="sessionExpirationSeconds" type="string" required={true}>Session expiration duration in seconds</ResponseField>
<ResponseField name="organizationId" type="string" required={true}>The organization ID this configuration applies to</ResponseField>
<ResponseField name="oauthClientIds" type="object" required={false}>Mapping of social login providers to their OAuth client IDs.</ResponseField>
<ResponseField name="oauthRedirectUrl" type="string" required={false}>OAuth redirect URL to be used for social login flows.</ResponseField>
<ResponseField name="otpAlphanumeric" type="boolean" required={false}>otpAlphanumeric field</ResponseField>
<ResponseField name="otpLength" type="string" required={false}>otpLength field</ResponseField>

<RequestExample>

```bash title="cURL"
curl --request POST \
  --url https://authproxy.turnkey.com/v1/wallet_kit_config \
  --header 'Accept: application/json' \
  --header 'Content-Type: application/json' \
  --header "X-Auth-Proxy-Config-Id: <string> (see Authorizations)" \
  --data '{}'
```

</RequestExample>
