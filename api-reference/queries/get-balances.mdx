---
title: "Get balances"
description: "Get balances of supported assets for an address on the specified network. Only non-zero balances are returned. This feature is in beta - please contact support for access."
---

import { Authorizations } from "/snippets/api/authorizations.mdx";
import { H3Bordered } from "/snippets/h3-bordered.mdx";
import { NestedParam } from "/snippets/nested-param.mdx";
import { EndpointPath } from "/snippets/api/endpoint.mdx";

<EndpointPath type="query" path="get_wallet_address_balances" />

<Authorizations />

<H3Bordered text="Body" />

<ParamField body="organizationId" type="string" required={true}>

Unique identifier for a given organization.
</ParamField>

<ParamField body="address" type="string" required={true}>

Address corresponding to a wallet account.
</ParamField>

<ParamField body="caip2" type="enum<string>" required={true}>

Enum options: `eip155:1`, `eip155:11155111`, `eip155:8453`, `eip155:84532`, `eip155:137`, `eip155:80002`
</ParamField>


<H3Bordered text="Response" />
A successful response returns the following fields:

<ResponseField name="balances" type="array" required={false}>
  List of asset balances
  <Expandable title="balances details">
    <NestedParam parentKey="balances" childKey="caip19" type="string" required={false}>
The caip-19 asset identifier
</NestedParam>
<NestedParam parentKey="balances" childKey="symbol" type="string" required={false}>
The asset symbol
</NestedParam>
<NestedParam parentKey="balances" childKey="balance" type="string" required={false}>
The balance in atomic units
</NestedParam>
<NestedParam parentKey="balances" childKey="decimals" type="number" required={false}>
The number of decimals this asset uses
</NestedParam>
<NestedParam parentKey="balances" childKey="display" type="object" required={false}>
      display field
      <Expandable title="display details">
        <NestedParam parentKey="balances.display" childKey="usd" type="string" required={false}>
USD value for display purposes only. Do not do any arithmetic or calculations with these, as the results could be imprecise.
</NestedParam>
<NestedParam parentKey="balances.display" childKey="crypto" type="string" required={false}>
Normalized crypto value for display purposes only. Do not do any arithmetic or calculations with these, as the results could be imprecise.
</NestedParam>

      </Expandable>
    </NestedParam>
<NestedParam parentKey="balances" childKey="name" type="string" required={false}>
The asset name
</NestedParam>

  </Expandable>
</ResponseField>

<RequestExample>

```bash title="cURL"
curl --request POST \
  --url https://api.turnkey.com/public/v1/query/get_wallet_address_balances \
  --header 'Accept: application/json' \
  --header 'Content-Type: application/json' \
  --header "X-Stamp: <string> (see Authorizations)" \
  --data '{
    "organizationId": "<string>",
    "address": "<string>",
    "caip2": "<eip155:1>"
}'
```

```javascript title="JavaScript"
import { Turnkey } from "@turnkey/sdk-server";

const turnkeyClient = new Turnkey({
  apiBaseUrl: "https://api.turnkey.com",
  apiPublicKey: process.env.API_PUBLIC_KEY!,
  apiPrivateKey: process.env.API_PRIVATE_KEY!,
  defaultOrganizationId: process.env.ORGANIZATION_ID!,
});

const response = await turnkeyClient.apiClient().getWalletAddressBalances({
  organizationId: "<string> (Unique identifier for a given organization.)",
  address: "<string> (Address corresponding to a wallet account.)",
  caip2: "<eip155:1>" // CAIP-2 chain ID (e.g., 'eip155:1' for Ethereum mainnet).
});
```

</RequestExample>
