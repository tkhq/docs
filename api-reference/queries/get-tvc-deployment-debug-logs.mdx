---
title: "Get TVC Deployment debug logs"
description: "Get a bounded window of application logs from a debug-mode TVC deployment. Returned lines are collected from every running replica and sorted by platform timestamp."
---

import { Authorizations } from "/snippets/api/authorizations.mdx";
import { H3Bordered } from "/snippets/h3-bordered.mdx";
import { NestedParam } from "/snippets/nested-param.mdx";
import { EndpointPath } from "/snippets/api/endpoint.mdx";

<EndpointPath type="query" path="get_tvc_deployment_debug_logs" />

<Authorizations />

<H3Bordered text="Body" />

<ParamField body="organizationId" type="string" required={true}>

Unique identifier for a given Organization.
</ParamField>

<ParamField body="deploymentId" type="string" required={true}>

Unique identifier for a given TVC Deployment. The deployment must be running in debug mode.
</ParamField>

<ParamField body="tailLines" type="number" required={false}>

Limit returned history to the last N lines per replica. If unset or zero, no tail-line limit is applied.
</ParamField>

<ParamField body="sinceSeconds" type="string" required={false}>

Return logs newer than this many seconds ago. If unset or zero, no since-time limit is applied. Useful for clients that poll to follow logs.
</ParamField>


<H3Bordered text="Response" />
A successful response returns the following fields:

<ResponseField name="entries" type="array" required={true}>
  Application log entries sorted by platform timestamp.
  <Expandable title="entries details">
    <NestedParam parentKey="entries" childKey="line" type="object" required={true}>
      line field
      <Expandable title="line details">
        <NestedParam parentKey="entries.line" childKey="content" type="string" required={true}>
One log line, exactly as the application printed it (without the trailing newline)
</NestedParam>
<NestedParam parentKey="entries.line" childKey="ts" type="object" required={false}>
      ts field
      <Expandable title="ts details">
        <NestedParam parentKey="entries.line.ts" childKey="seconds" type="string" required={true}>
seconds field
</NestedParam>
<NestedParam parentKey="entries.line.ts" childKey="nanos" type="string" required={true}>
nanos field
</NestedParam>

      </Expandable>
    </NestedParam>

      </Expandable>
    </NestedParam>
<NestedParam parentKey="entries" childKey="replicaLabel" type="string" required={true}>
Public replica label that produced this log line, for example 'replica 2/3'.
</NestedParam>

  </Expandable>
</ResponseField>

```bash title="cURL"
  --header "X-Stamp: <string> (see Authorizations)" \
  --data '{

    "tailLines": "<number>",
    "sinceSeconds": "<string>"

```

```javascript title="JavaScript"

});
```
