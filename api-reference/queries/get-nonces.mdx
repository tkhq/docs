---
title: "Get nonces"
description: "Get nonce values for an address on a given network. Can fetch the standard on-chain nonce and/or the gas station nonce used for sponsored transactions."
---

import { Authorizations } from "/snippets/api/authorizations.mdx";
import { H3Bordered } from "/snippets/h3-bordered.mdx";
import { NestedParam } from "/snippets/nested-param.mdx";
import { EndpointPath } from "/snippets/api/endpoint.mdx";

<EndpointPath type="query" path="get_nonces" />

<Authorizations />

<H3Bordered text="Body" />

<ParamField body="organizationId" type="string" required={true}>

Unique identifier for a given Organization.
</ParamField>

<ParamField body="address" type="string" required={true}>

The Ethereum address to query nonces for.
</ParamField>

<ParamField body="caip2" type="enum<string>" required={true}>

Enum options: `eip155:1`, `eip155:11155111`, `eip155:8453`, `eip155:84532`, `eip155:137`, `eip155:80002`
</ParamField>

<ParamField body="nonce" type="boolean" required={false}>

Whether to fetch the standard on-chain nonce.
</ParamField>

<ParamField body="gasStationNonce" type="boolean" required={false}>

Whether to fetch the gas station nonce used for sponsored transactions.
</ParamField>


<H3Bordered text="Response" />
A successful response returns the following fields:

<ResponseField name="nonce" type="string" required={false}>The standard on-chain nonce for the address, if requested.</ResponseField>
<ResponseField name="gasStationNonce" type="string" required={false}>The gas station nonce for sponsored transactions, if requested.</ResponseField>

<RequestExample>

```bash title="cURL"
curl --request POST \
  --url https://api.turnkey.com/public/v1/query/get_nonces \
  --header 'Accept: application/json' \
  --header 'Content-Type: application/json' \
  --header "X-Stamp: <string> (see Authorizations)" \
  --data '{
    "organizationId": "<string>",
    "address": "<string>",
    "caip2": "<eip155:1>",
    "nonce": "<boolean>",
    "gasStationNonce": "<boolean>"
}'
```

```javascript title="JavaScript"
import { Turnkey } from "@turnkey/sdk-server";

const turnkeyClient = new Turnkey({
  apiBaseUrl: "https://api.turnkey.com",
  apiPublicKey: process.env.API_PUBLIC_KEY!,
  apiPrivateKey: process.env.API_PRIVATE_KEY!,
  defaultOrganizationId: process.env.ORGANIZATION_ID!,
});

const response = await turnkeyClient.apiClient().getNonces({
  organizationId: "<string> (Unique identifier for a given Organization.)",
  address: "<string> (The Ethereum address to query nonces for.)",
  caip2: "<eip155:1>" // CAIP-2 chain ID (e.g., 'eip155:1' for Ethereum mainnet).,
  nonce: true // Whether to fetch the standard on-chain nonce.,
  gasStationNonce: true // Whether to fetch the gas station nonce used for sponsored transactions.
});
```

</RequestExample>

<ResponseExample>

```json 200
{
  "nonce": "<string>",
  "gasStationNonce": "<string>"
```
