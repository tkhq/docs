<EndpointPath type="query" path="get_attestation" />

<Authorizations />

<H3Bordered text="Body" />

<ParamField body="organizationId" type="string" required={true}>

Unique identifier for a given organization.
</ParamField>

<ParamField body="enclaveType" type="string" required={true}>

The enclave type, one of: ump, notarizer, signer, evm-parser.
</ParamField>


<H3Bordered text="Response" />
A successful response returns the following fields:

<ResponseField name="attestationDocument" type="string" required={true}>Raw (CBOR-encoded) attestation document.</ResponseField>
