<H3Bordered text="Overview" />

<ul>
  <li>**API subject to change**</li>
</ul>

Signs and submits an ERC20 ``transfer(address,uint256)`` as an Ethereum transaction using a Turnkey-managed (embedded) wallet.

This is a convenience wrapper around ``ethSendTransaction``:

<ul>
  <li>Encodes ERC20 transfer calldata.</li>
  <li>Sends a transaction to the token contract.</li>
  <li>Returns a ``sendTransactionStatusId`` for polling with ``pollTransactionStatus``.</li>
</ul>
