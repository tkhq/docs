<H3Bordered text="Overview" />

**API subject to change**

Polls Turnkey for the final result of a previously submitted transaction.

This function repeatedly calls ``getSendTransactionStatus`` until the transaction reaches a terminal state.

Terminal states:

<ul>
  <li>**COMPLETED** or **INCLUDED** â†’ resolves with chain-specific transaction details</li>
  <li>**FAILED** rejects with an error</li>
</ul>

Behavior:

<ul>
  <li>Queries Turnkey every 500ms.</li>
  <li>Stops polling automatically when a terminal state is reached.</li>
  <li>Returns the full status payload from Turnkey.</li>
  <li>When available, Ethereum transaction details are exposed at ``resp.eth.txHash``.</li>
</ul>
