<H3Bordered text="Overview" />

Handles signing and sending an ERC20 ``transfer(address,uint256)`` transaction.

<ul>
  <li>Submits an ERC20 transfer intent to Turnkey for signing and execution.</li>
  <li>Automatically polls the transaction status until it reaches a terminal state.</li>
  <li>Displays a modal showing progress and a success page upon completion.</li>
</ul>
