<H3Bordered text="Overview" />

Refreshes the wallets state for the current user session.

<ul>
  <li>This function fetches the latest list of wallets associated with the current session or user,</li>
</ul>

and updates the ``wallets`` state variable in the provider.

<ul>
  <li>If a ``stampWith`` parameter is provided, it will use that stamper to fetch the wallets</li>
</ul>

(supports Passkey, ApiKey, or Wallet stampers for granular authentication control).

<ul>
  <li>Automatically handles error reporting via the configured callbacks.</li>
  <li>Typically used after wallet creation, import, export, account changes, or authentication</li>
</ul>

to ensure the provider state is up to date.

<ul>
  <li>If no wallets are found, the state will be set to an empty array.</li>
</ul>
