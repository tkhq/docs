<H3Bordered text="Overview" />

Handles the fiat onramp process for converting fiat currency into crypto and funding a wallet.

<ul>
  <li>Initializes a fiat onramp transaction with a specified provider (e.g., Coinbase or MoonPay).</li>
  <li>Opens the provider flow in a new window and polls the transaction status until completion.</li>
  <li>Displays a modal to show progress and success state.</li>
  <li>Supports both sandbox and production modes.</li>
</ul>
