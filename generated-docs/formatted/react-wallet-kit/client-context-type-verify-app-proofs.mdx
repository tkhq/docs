<H3Bordered text="Overview" />

Verifies a list of app proofs against their corresponding boot proofs.

<ul>
  <li>This function iterates through each provided app proof, fetches the corresponding boot proof, and verifies the app proof against the boot proof.</li>
  <li>If any app proof fails verification, an error is thrown.</li>
</ul>

<ParamField body="params" type='VerifyAppProofsParams' required={true} />

A successful response returns the following fields:

<ResponseField name="returns" type="void" required={true}>
  A promise that resolves when all app proofs have been successfully verified.
</ResponseField>
